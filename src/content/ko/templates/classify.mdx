> 클릭만으로 무엇이든 분류하세요. 코드 없이, 직관만으로.

이 템플릿은 아주 단순한 질문에서 시작됐습니다.  
**"전처리부터 모델 선택까지, 분류 전 과정을 자동화할 수 있다면 어떨까?"**  

namdarine Classify는 단 한 줄의 코드 없이  
데이터 속 패턴을 찾아내는 분류 워크플로우를 제공합니다.  
데이터셋을 업로드하면, 그 다음은 시스템이 알아서 처리합니다 —  
전처리, 모델 선택, PDF 리포트까지 모두 자동입니다.  

---

### 🔧 주요 기능

- **데이터 업로드**  
  CSV 파일 업로드만으로 시작 -  
  숫자형, 범주형, 문장 (텍스트)까지 자동으로 인식됩니다.  

- **전처리 자동화**  
  결측치 처리, 인코딩, 스케일링까지 전 과정 자동 수행  

- **스마트 모델 선택**  
  나이브 베이즈, 의사결정나무, 랜덤 포레스트, 로지스틱 회귀 중  
  → ROC-AUC 기준 최적의 모델 자동 선택  

- **PDF 리포트 생성**  
  주요 지표, 성능 시각화, 인사이트가 담긴 깔끔한 PDF 보고서 제공  

- **다운로드 옵션**  
  예측 결과는 CSV로, 학습된 모델은 재사용 가능하도록 다운로드 제공
